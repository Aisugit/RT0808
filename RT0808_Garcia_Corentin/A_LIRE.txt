Les deux dossiers dans ce répertoire contiennent respectivement les codes concernant WebRTC et WebSocket.

Pour WebRTC, simplement lancer la page index.html dans ce dernier permettra d'acceder aux autres pages de test.

Concernant WebSocket, on trouva dans le dossier deux répertoires contenant dans l'un, un exmeple d'utilisation sans naviguateur, et dans l'autre un exemple d'utilisation avec naviguateur.
Dans les deux cas, le serveur doit être exécuter avant le client et la page web représentant le client dans l'exemple web peut être ouverte plusieurs fois pour avoir plusieurs clients de connecter à l'application.
